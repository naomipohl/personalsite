Python/Flask

Graph Property-Checker: Heroku link

28 Nov, 2017

*The app might take ~15 seconds to load (the server goes to sleep after a few minutes of inactivity).*

In my discrete math class this semester, instead of a second midterm we were assigned an open-ended project.
In class, we'd learned how to determine whether graphs have Euler paths/cycles and Hamiltonian circuits by hand, as well as how to tell whether a graph is bipartite. I wanted to learn how a computer would approach these problems, so a few classmates and I put together this Flask app. We used python for the back-end algorithms and javascript/HTML for the front-end jazz.
With our app, a user can draw a graph and click a button to find out whether the graph has certain properties. This was my first project with Flask, and yes, it really is as straightforward as they say it is!
